1. Fix the black background

The black background is part of the PNG asset, not a CSS background.

Create a cleaned transparent version of the Level 1 image.

Input:
assets/level1_garage_base.png

Output:
assets/level1_garage_base_clean.png

Remove the black/near-black background outside the room and preserve the room itself.

Implementation options:

* If the project has image tooling, use it.
* Otherwise create a small Node or Python helper script to convert near-black pixels to alpha.
* Do not manually hide the black area with CSS.
* Do not use mix-blend-mode as a fake fix.

Background removal rule:

* pixels with very low RGB values outside the room should become transparent
* keep dark interior shadows/details intact as much as possible
* use a conservative threshold, for example RGB < 12 or RGB < 18
* if needed, use flood-fill from image borders so only connected black background is removed, not dark details inside the room

Preferred algorithm:

* load image
* flood-fill from all border pixels that are near-black
* set those connected pixels to alpha 0
* save as PNG with transparency

Use the cleaned file in the scene:
level1_garage_base_clean.png
